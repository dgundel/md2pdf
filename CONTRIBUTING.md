# Contributing to md2pdf

- **Bugs & Ideen:** Bitte ein [Issue](https://github.com/dgundel/md2pdf/issues) öffnen.
- **Pull Requests:** Gerne willkommen. Für größere Änderungen zuerst kurz im Issue besprechen.
- **Tests:** Vor dem PR `pytest tests/ -v` ausführen. Neue Features am besten mit Tests ergänzen.
- **Code:** Python 3.10+, Stil optional mit `ruff check md2pdf/` prüfen.
